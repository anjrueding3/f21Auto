# f21Auto

### Reads data from PDFs to Excel File

#### Dependencies

##### PyPDF2
`pip install PyPDF2`

##### openpyxl
`pip install openpyxl`


# To Use Code:

##### 1. Clone 

##### 2. Navigate to project src code folder in Terminal

##### 3. Run:
`./main.py __directoryName__ __targetExcelFile__`
